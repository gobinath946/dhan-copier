## Optimization Tips

### **If Win Rate Too Low (<60%):**
- Increase master score threshold (75 → 80)
- Increase confidence threshold (8 → 9)
- Increase agreement requirement (7 → 8)

### **If Too Few Trades (<3/day):**
- Decrease master score threshold (75 → 70)
- Decrease confidence threshold (8 → 7)
- Decrease agreement requirement (7 → 6)

### **If AI Calls Too Slow:**
- Use faster model (gpt-4o-mini)
- Reduce ensemble size (5 → 3 for entry)
- Cache algorithm results
